# coco_telegram_ux_polish

Use this skill for Telegram display improvement only.

Allowed:
- Chinese wording polish.
- Better status cards.
- More readable help menu.
- Mobile-friendly report summaries.
- Clear blocked-action messages.

Forbidden:
- changing runtime guard
- changing deploy/rollback behavior
- weakening LOCAL_ONLY
- adding real API calls
- reading private config
- changing trading/VPS logic
