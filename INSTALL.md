# Install guide

## Upload to GitHub

Use `scripts/publish_to_github.ps1`.

Use a fine-grained GitHub personal access token with repository access only to `coco101-weitdo/coco-codex-skills` and Contents read/write permission.

Do not paste the token into ChatGPT. Paste it only into the local PowerShell prompt.

## Install in Codex

Use `skill-installer` with GitHub directory URLs, one skill at a time.
